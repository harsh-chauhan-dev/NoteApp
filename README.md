# Personal Knowledge Management Platform

A full-stack web application designed to help users securely capture, organize, and manage personal information in a centralized workspace.

The platform addresses the common challenge of scattered information by providing a structured system for storing, updating, and retrieving user-generated content. It incorporates secure authentication, role-based access control principles, and efficient data management workflows to ensure privacy and usability.

## Key Features

* Secure user authentication using JWT and HTTP-only cookies
* User-specific data isolation and authorization
* Create, update, retrieve, and delete content records
* RESTful API architecture for scalable communication
* PostgreSQL-powered persistent data storage
* Responsive and intuitive user interface
* Input validation and centralized error handling
* Protected routes and session management

## Technical Highlights

* Built with PostgreSQL, Express.js, React.js, and Node.js (PERN Stack)
* Implemented authentication and authorization mechanisms
* Designed relational database schemas and optimized queries
* Developed reusable frontend components and API integrations
* Applied secure password hashing using bcrypt
* Followed modular backend architecture and REST API best practices

## Objective

This project demonstrates practical experience in full-stack software development, backend architecture, database management, authentication systems, and secure web application design.
